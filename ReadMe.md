# Sales Performance Analysis Dashboard (Excel)

## Overview
This project presents a Sales Performance Dashboard created using Microsoft Excel to analyze key business metrics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Key Insights
- Total Sales
- Profit Analysis
- Regional Sales Performance
- Category-wise Sales
- Sales Trends

## Features
- Interactive slicers
- Dynamic charts
- KPI tracking
- Clean dashboard layout
